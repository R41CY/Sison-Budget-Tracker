# Sison Financial – Budget & Profit Tracking System

A **web-based financial tracking system** built for our family business to monitor daily income, expenses, and overall profitability.

The system simplifies financial monitoring by automatically calculating profit, margins, and production costs without relying on spreadsheets.

## Features

### Transaction Tracking
Record daily **income and expenses** for accurate financial monitoring.

### Automatic Profit Calculations
Instantly calculates **profit and profit margins** based on recorded transactions.

### Recipe Costing System
Calculate the **production cost of food items** by tracking ingredient expenses.

### Financial Dashboard
View a clear summary of **business performance and financial status**.

### Daily & Weekly Reports
Generate reports to monitor **short-term financial performance**.

### Data Backup & Export
Backup and export financial data for **safety and portability**.

## Purpose

This system was created to help small businesses **track profitability, manage expenses, and understand their financial performance** without needing complex accounting software or spreadsheets.
