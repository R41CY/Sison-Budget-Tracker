Sison Financial – Budget & Profit Tracking System

A web-based financial tracker I built for our family business to monitor daily income, expenses, and profitability.

Features:
• Transaction tracking for income and expenses
• Automatic profit and margin calculations
• Recipe costing system to calculate food production costs
• Dashboard showing financial performance
• Daily and weekly financial reports
• Data backup and export system

The system helps track business profitability and automate financial calculations without needing spreadsheets.
